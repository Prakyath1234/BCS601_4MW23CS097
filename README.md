## Project Description

Student Utility App is a simple web application developed using **Python Flask** and deployed on the **Render Cloud Platform**.

The application allows users to:
- Enter two numbers and calculate HCF and LCM
- Enter a string and reverse it
- Generate factorial values from 4 to 8

This project demonstrates:
- Web application development using Flask
- Cloud deployment using Render
- Version control using GitHub

---

## Features

### Number Operations
- Calculate **HCF (Highest Common Factor)** of two numbers
- Calculate **LCM (Least Common Multiple)** of two numbers

### String Operation
- Reverse user-entered string

### Factorial Generator
Displays factorial values for:
- 4! = 24
- 5! = 120
- 6! = 720
- 7! = 5040
- 8! = 40320
